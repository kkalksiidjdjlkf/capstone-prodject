# Secure Offline RAG System for Enterprise AI Assistants

A Retrieval-Augmented Generation (RAG) system designed for Korea Renewable Energy and Power Solutions (KREPS). This system is engineered for high-security, offline processing of technical documentation with a focus on accuracy and data sovereignty.

## 📋 Project Overview
This project provides an AI-powered assistant capable of answering complex queries about internal technical documents (PDF, CSV, TXT) without an internet connection. It leverages hybrid search and local Large Language Models (LLMs) to ensure that sensitive data never leaves the corporate perimeter.

### Core Technology Stack:
- **LLM:** Llama 3.1 8B (via Ollama)
- **Embeddings:** BAAI/BGE-M3 (High-performance multilingual)
- **Reranker:** BAAI/BGE-Reranker-v2-m3
- **Vector Store:** FAISS
- **Hybrid Search:** Combined BM25 (Keyword) and Vector (Semantic) search

## 🏗 System Architecture
The system follows a modular RAG pipeline designed for scalability and transparency:



1. **Ingestion & Preprocessing**: Documents are parsed (using Docling for high-quality PDF extraction) and split into optimized chunks.
2. **Dual Encoding**: 
   - **Dense**: Semantic vectors generated by BGE-M3.
   - **Sparse**: Keyword indexing using BM25.
3. **Hybrid Retrieval**: Queries are processed through both indexes to ensure both keyword accuracy and semantic context.
4. **Reranking Stage**: The Top-K results are re-evaluated by a Cross-Encoder to filter out noise and improve relevance.
5. **Generation**: Llama 3.1 generates a comprehensive response based strictly on the retrieved context.

## 🚀 Key Features
- **Total Privacy**: 100% offline deployment, no external API calls.
- **Multilingual Support**: Optimized for Korean and English technical content.
- **Enterprise-Grade Accuracy**: Reranking stage ensures "Quality Over Speed."
- **Hardware Agnostic**: Supports NVIDIA CUDA (Server) and Apple Metal/MPS (MacBook).
